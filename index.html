Tic Tac Toebeans

I made this Tic Tac Toe game as a coding challenge to make aspects of the program object oriented. I used a constructor function for both the individual tiles and the entire game board. I made the tiles into button switches to give a visual cue that they cannot be pressed after one has been selected or when the game is completed.
